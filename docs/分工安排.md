下面按当前 CodePilot 项目结构分工：刘子恒负责前端；郑嘉娴、钟经添、曾嘉睿、周子涵负责后端。两个 Sprint，每个 Sprint 2 天。每个 Sprint 结束都能进行功能测试，两个 Sprint 后完成核心闭环：前端提交任务、后端创建任务、Agent 执行、工具调用、权限确认、日志记录、历史回放。

**总体分工原则**

| 成员 | 方向 | 模块边界 | 目标 |
|---|---|---|---|
| 刘子恒 | 前端 | Web 前端交互与接口对接 | 完成任务输入、状态展示、历史记录、权限确认、日志/输出展示 |
| 郑嘉娴 | 后端 | API 接口层 + 服务编排 | 负责 REST API、任务/会话/工作区接口入口 |
| 钟经添 | 后端 | SQLite 存储 + 日志回放 | 负责数据库表、Repository、历史日志、回放数据 |
| 曾嘉睿 | 后端 | Agent Core + LLM 调用 | 负责 Agent 执行流程、任务规划、上下文、AI 调用 |
| 周子涵 | 后端 | 工具系统 + 权限安全 + 事件流 | 负责 file/shell/git 工具、权限判断、SSE 事件推送 |
